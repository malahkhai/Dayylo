# Skill: Premium Mobile UI Design

## Description
AI agent must generate mobile app UI with a premium, Apple-style design. All spacing, typography, padding, colors, and components must follow Human Interface Guidelines (Apple HIG) and maintain a consistent, high-end look.

## References
- Apple HIG: https://developer.apple.com/design/human-interface-guidelines/
- Material Design (optional for Android): https://m3.material.io/
- Example UI kits: https://www.figma.com/community

## Typography
- Primary: San Francisco Pro (iOS), fallback system font
- Headings:
  - H1: 32pt Bold
  - H2: 28pt Semibold
  - H3: 24pt Semibold
- Body:
  - Body1: 16pt Regular, line-height 1.4x
  - Body2: 14pt Regular, line-height 1.4x
- Emphasis: Italics for secondary info, Medium weight for actionable text

## Spacing & Layout
- Base unit: 8pt
- Small gap: 8pt, Medium: 16pt, Large: 24-32pt
- Standard container padding: 16pt
- Safe Area compliance required for top/bottom margins
- Avoid overcrowding: white space is key

## Components
- Buttons:
  - Rounded corners 12pt
  - Padding: 12pt vertical, 20pt horizontal
  - Primary color: #007AFF
  - Secondary color: #34C759
- Cards:
  - Padding: 16pt, Rounded corners 12pt, subtle shadow
- Inputs:
  - Border: 1pt, Corner radius: 12pt
  - Padding: 12pt horizontal, 8pt vertical
- Lists:
  - Consistent row height (60-80pt)
  - 16pt spacing between items

## Colors
- Background: #FFFFFF
- Text: #000000 / #1C1C1E
- Accent: #007AFF, #34C759
- Shadows: rgba(0,0,0,0.05) for subtle depth

## Interactions & Motion
- Smooth fade, scale, or slide transitions
- Tap feedback via slight scale-down/highlight
- Avoid abrupt jumps or heavy animations

## Output Guidelines
- Always follow above typography, spacing, and component rules
- Keep a clean, consistent premium aesthetic
- Reference Apple HIG where needed
- Apply skill to all screens, including onboarding, dashboard, and forms
